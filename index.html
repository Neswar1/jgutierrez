Sistema de gestión de entrenamiento

Aplicación desktop desarrollada para la gestión y control de horas de entrenamiento para operadores en un entorno industrial.

Características principales:

  • Interfaz desarrollada con PyQT5 Designer
  • Base de datos SQLite para almacenamiento de información
  • Sistema de autenticación con dos tipos de usuarios (Admin/Operador)
  • Creación y gestión de usuarios
  • Asignación y creación de distintos procesos con sistema confirmación Admin - Operador

Tecnologías utilizadas:

Python SQLite PyQT5

Interfaz de login

Ventana de administrador

Creación y manejo de procesos

El sistema permite crear nuevos procesos que constan de 2 tipos de dato, "Proceso" y "Horas Totales" en este caso el Proceso representa al nombre asignado y las Horas Totales ncesarias para completar dicho proceso

Una vez creado el proceso, este se puede asignar a un usuario, el cual podrá confirmar las horas que ha trabajado en dicho proceso.

Gestión de Usuarios

El administrador se encarga de crear,modificar y gestionar todos los usuarios (pueden existir distintos adminstradores), como se muestra en la imagen podemos ver los datos necesarios para crear un nuevo usuario o en su caso modificar alguno ya existente, también podemos visualizar todos los usuarios en la tabla de la derecha.

Podemos seleccionar un usuario cualquiera y darle doble click, esto nos desplegara una nueva ventana donde podemos observar la información detallada de dicho usuario al igual que asigarle procesos nuevos o eliminarlos, tambien confirmar horas pendientes realizando doble click sobre dicho proceso.

Cuando un usuario se encuentra de color naranja significa que tiene horarios pendientes por confirmar.

Ventana de administrador

Ventana individual de usuario

Desde esta ventana el administrador puedo gestionar las horas y procesos asignados a un usuario, también es posible monitorear el avance.

Mediante el combobox del centro superior podemos seleccionar los procesos que deseemos agregar, seleccionado y pulsando el boton de Asignar certificado, para eliminar alguno debes selecionarlo de la tabla y clickear Eliminar certificado.

Podemos darle doble click sobre el proceso que necesita horas de confirmación al hacerlo sobre alguno que no necesite no pasa nada, pero si tiene horas pendientes se desplegara un mensaje donde nos preguntara si queremos confirmar las horas ingresas por el usuario (nos dirá cuantas horas ingreso).

Interfaz del usuario

Para ingresar como usuario normal (operador) simplemente ingresamos con unas credenciales activas de un usuario creado por el administrador, lo que veremos será la interfaz del usuario desde la cual podemos observar los procesos que tenemos asignados y de igual manera poder crear una solicitud en un proceso.

Lo que debemos de hacer es darle click en la tabla al proceso que deseemos agregar horas de entrenamiento y pulsar el boton de Actualizar progreso, ahi se nos desplegara un input que nos solicitara las horas de practica realizadas.
